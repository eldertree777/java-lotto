## 프로그래밍 요구사항
- 핵심 로직과 UI 로직을 분리한다.
- indent(들여쓰기) depth를 2단계에서 1단계로 줄여라.
- 메소드의 크기가 최대 15라인을 넘지 않도록 구현한다.
- method가 한 가지 일만 하도록 최대한 작게 만들어라.
- else를 사용하지 마라.
- 모든 로직에 단위 테스트를 구현한다.
- 자바 코드 컨벤션을 지키면서 프로그래밍 하기.
